# 🌦️ Retro Weather

A retro-styled weather lookup app — enter a city and get current weather conditions displayed in a nostalgic interface.

---

## 🖥️ About

A simple, clean weather app where users can search any city and view current conditions, styled with a retro visual theme.

---

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Displays current weather conditions
- 🎨 Retro-styled UI

---

## 🛠️ Built With

- **Frontend:** HTML, CSS, JavaScript
- **Weather Data:** Weather API *(update with the specific API you used, e.g. OpenWeatherMap)*

---

## ⚙️ How It Works

1. User enters a city name into the search form
2. The app fetches current weather data for that city
3. Results are displayed dynamically in the weather card

---

## 🚀 Getting Started

```bash
git clone <your-repo-url>
cd retro-weather
```

Open `index.html` directly in your browser — no build step required.
